# GLRenderer


This is the foundations of an opengl renderer, originally meant for a game engine.
![Alt text](./demo-pict.png?raw=true "demo")

<h6>-this project uses the following libraries:</h6>
	**included in source:**
		*-glad
		*-stb_image
		*-imgui
		*-glm
	**not included in source:**
		*-assimp
		*-glfw3
